Three-step distributed authentication flow

1Login (AS Exchange)

Authenticate with username and password. The password is never sent to the API server.

2Get Service Ticket (TGS Exchange)

Request a service ticket for the API server using your TGT.

3Access Protected Resource (AP Exchange)

Call a protected API endpoint using your service ticket.

Tips: Try clicking "Call" twice to see replay attack prevention in action.
